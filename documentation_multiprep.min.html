Vue d’ensemble

MultiPrep est une application Windows PySide6 qui transforme des PDF, documents Word et images en un montage PDF ordonné. La version 2.0.0 démarre en mode Gmail et conserve un mode classique pour les fichiers locaux et applications de bureau.

EntréesPDF, DOC, DOCX, JPG, JPEG et PNG.
TraitementAperçus, sélection, rotation, réorganisation et séparateurs.
SortieAAAA-MM-JJ_NOM-Prenom_P.pdf.

Modes d’utilisation

Mode Gmail — par défautPièces jointes glissées depuis Gmail dans un navigateur. Les images présentes dans le corps du mail sont copiées puis collées avec Ctrl+V ou le menu Coller.
Mode classiqueFichiers venant de l’Explorateur, du Bureau ou d’une application de bureau qui expose un vrai fichier, dont Outlook bureau.

Important : le helper Gmail est un composant Windows natif intégré à la grande zone. Il sert à recevoir les fichiers virtuels que Qt ne peut pas toujours extraire seul. Il n’ouvre pas de fenêtre séparée.

Chaînes d’import

Fichier local

Qt transmet un chemin local → validation de l’extension → création du modèle → ajout immédiat à la grille → miniature en arrière-plan.

Pièce jointe Gmail

Le navigateur transmet un objet virtuel → le helper Windows récupère son contenu asynchrone → écrit un fichier temporaire → publie un manifeste → Python importe le fichier.

Image dans le corps du mail

Le presse-papiers est lu comme bitmap, image Qt, HTML/Base64 ou URL d’image → conversion en page PDF → ajout au montage.

Document Word

Microsoft Word s’exécute sans interface via PowerShell COM → export PDF → import par le moteur PDF commun.

Moteur d’affichage rapide

  • aucun rendu de miniature pendant la création des modèles ;
  • delegate Qt léger au lieu d’un widget complet par page ;
  • jusqu’à quatre lots de miniatures en parallèle ;
  • ouverture groupée d’un même PDF ;
  • ajout incrémental sans reconstruire les pages existantes ;
  • cache local des aperçus.

Référence mesurée : 500 pages injectées en environ 10–12 ms ; 120 miniatures réelles en environ 0,63 seconde sur le poste de développement.

Compilation

Compiler d’abord assets/GmailDropHelper.exe depuis le source C#, puis lancer PyInstaller avec MultiPrep.spec.

python -m PyInstaller --noconfirm --clean `
  --workpath "$env:TEMP\multiprep-build" `
  --distpath dist MultiPrep.spec

Le livrable officiel est dist/MultiPrep.exe. Les logos 2.0.0 et le helper Gmail sont intégrés par le fichier spec.

Tests et validation

python -m compileall -q multiprep tests
python -m unittest discover -s tests -v
git diff --check

Les tests couvrent les chemins locaux, Word, les images Qt/HTML, les fichiers virtuels de navigateur et la compatibilité Outlook.
